# 📈 Sales Forecasting Dashboard

## Project Overview

This project forecasts future Superstore sales using Machine Learning and Time Series Forecasting techniques. It compares multiple forecasting models and identifies the best-performing model based on evaluation metrics.

---

## Dataset

- Superstore Sales Dataset

---

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Statsmodels
- Prophet
- XGBoost
- Streamlit

---

## Models Used

- SARIMA
- Prophet
- XGBoost

---

## Evaluation Metrics

| Model | MAE | RMSE | MAPE |
|-------|------:|------:|------:|
| SARIMA | 16295.76 | 21413.03 | 22.49% |
| Prophet | **10128.56** | **14561.39** | **14.33%** |
| XGBoost | 22902.48 | 26329.60 | 33.32% |

---

## Best Model

**Prophet**

MAPE: **14.33%**
